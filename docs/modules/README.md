# Module Documentation

This folder documents how to design, build, register, and maintain a Sky Fundi module. For the architectural contract modules must satisfy, see [`../architecture/module-system.md`](../architecture/module-system.md). This folder is the practical, how-to companion to that contract.

- [`module-development-guide.md`](module-development-guide.md) — step-by-step guide to scaffolding a new module
- [`module-lifecycle.md`](module-lifecycle.md) — install/enable/disable/update/remove operational detail
